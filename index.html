Hubert-basket.

Au cours de mon alternance chez Chausport, j'ai travaillé sur un projet nommé Hubert-basket. Ce projet vise a mettre en place un système de click & collect et d'e-réservation. Durant ce projet, j'ai pu pratiquer les méthodes de gestion de projet agiles. Les technologies utilisées sont le C# avec le framework .NET, le JavaScript avec le framework Angular, le SQL avec les systèmes de gestion de base de données Microsoft SQL Server et MariaDB.
